# Backend 

## Quickstart
### Installation 
```
$ mkdir env 
$ python3 -m venv env 
$ source env/bin/activate 
$ pip install -r requirements.txt
```

### Migrate Set Up 
```sh
$ py manage.py makemigrations 
$ py manage.py migrate 
```

### Running the Application 
```sh 
$ py manage.py runserver 
```

### Additional Note 
Make sure postgres is installed
